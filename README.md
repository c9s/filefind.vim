
FileFind.vim
============

INSTALL
=======

    $ make install

    $ cat function.sh >> ~/.zshrc
    $ cat function.sh >> ~/.bashrc

USAGE
=====

    $ fv pattern 

    $ f pattern | vim -

MAPPING
=======

- `<ENTER>` - open file
- `D`       - delete file
- `G`       - grep file
- `?`       - show help message
